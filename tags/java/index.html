#####2. 删除tomcat目录webapps下的项目文件

#####3. 上传war包到webapps目录下

#####4. 运行bin目录下的catalina.sh，即可以带日志形式启动tomcat并解压war包

#####5. 运行bin目录下的startup.sh即可启动tomcat
